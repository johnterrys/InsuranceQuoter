# Insurance Quoter
Azure Cloud Native Proof of Concept comprising 
- ASP.NET Core Blazor SPA using Fluxor
- OpenId Connect with Identity Server
- ASP.NET Core Web API
- ASP.NET Core SignalR
- Cosmos Db 
- Azure Service Bus running NServiceBus
